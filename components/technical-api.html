API Reference

AHP Mod 2.0 exposes a powerful yet intuitive API that allows developers to integrate the protocol into any application with minimal effort:

POST /api/v1/process Core
Process a human message and return an AI response using the AHP protocol. This endpoint handles the complete translation process across all protocol layers.
Parameter
Type
Description
message
string
The human message to process through the protocol
context
object
Optional context information to enhance intent recognition
model
string
Target AI model identifier (e.g., "gpt-4", "claude-3")
options
object
Advanced configuration options for protocol behavior
Response
{
  "id": "msg_12345",
  "processed_message": "Processed human message",
  "ai_response": "AI system response",
  "intent": {
    "primary": "query",
    "confidence": 0.95
  },
  "model_used": "gpt-4",
  "processing_time": 0.45
}
GET /api/v1/models
List all available AI models that can be used with the protocol, including capabilities and compatibility information.
Response
{
  "models": [
    {
      "id": "gpt-4",
      "provider": "OpenAI",
      "capabilities": ["text", "reasoning", "code"],
      "compatibility_score": 0.98
    },
    {
      "id": "claude-3",
      "provider": "Anthropic",
      "capabilities": ["text", "reasoning", "analysis"],
      "compatibility_score": 0.96
    }
  ]
}
POST /api/v1/analyze
Analyze human communication to extract intent and context without generating an AI response. Useful for preprocessing and understanding user inputs.
